Operation Signature:
   getSecurePassword(salt : String, passwordToHash : String) : String
	Where getSecurePassword is a public instance method of the PasswordHasher class
	where salt is a String containing a randomly generated sequence and passwordToHash
	is the password inputted by the user. Method returns the password encrypted using the 
	SHA-512 encryption algorithm using the salt
 Preconditions: 
	password must not be null and must have more than 6 characters.
	Salt must be generated by SecureRandom;
 Postconditions:
	no data is stored
 Invariant:
        None
 Frame Condition:
     All data in the calling class and the called object will remain unchanged.
     This method serves as a service provider to generate and return the hashed password only
